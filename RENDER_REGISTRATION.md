# Регистрация на Render.com

## Способ 1: Через GitHub (Рекомендуется)

### Шаг 1: Регистрация на GitHub (если еще нет аккаунта)

1. Зайдите на [github.com](https://github.com)
2. Нажмите "Sign up" (вверху справа)
3. Заполните форму:
   - **Username**: ваш никнейм
   - **Email**: ваш email
   - **Password**: создайте пароль
4. Решите капчу и нажмите "Create account"
5. Подтвердите email (проверьте почту)

### Шаг 2: Регистрация на Render через GitHub

1. Зайдите на [render.com](https://render.com)
2. Нажмите **"Get Started for Free"** (большая кнопка)
3. Выберите **"Continue with GitHub"**
4. Разрешите доступ Render к вашему GitHub аккаунту
5. Готово! Вы зарегистрированы

**Преимущества:**
- ✅ Быстрая регистрация
- ✅ Автоматическое подключение репозиториев
- ✅ Не нужно подтверждать email отдельно

---

## Способ 2: Через Email

### Шаг 1: Регистрация

1. Зайдите на [render.com](https://render.com)
2. Нажмите **"Get Started for Free"**
3. Выберите **"Sign up with Email"**
4. Заполните форму:
   - **Email**: ваш email
   - **Password**: создайте пароль (минимум 8 символов)
5. Нажмите "Sign up"

### Шаг 2: Подтверждение email

1. Проверьте почту
2. Найдите письмо от Render
3. Нажмите на ссылку подтверждения
4. Готово! Вы зарегистрированы

### Шаг 3: Подключение GitHub (опционально, но рекомендуется)

1. В Dashboard Render нажмите на ваш профиль (вверху справа)
2. Выберите **"Account Settings"**
3. Перейдите в **"Connected Accounts"**
4. Нажмите **"Connect GitHub"**
5. Разрешите доступ

**Зачем это нужно:**
- Автоматическое подключение репозиториев
- Удобный деплой из GitHub

## Бесплатный план Render

✅ **Что включено:**
- 750 часов работы в месяц (достаточно для постоянной работы)
- HTTPS автоматически
- Автоматический деплой из GitHub
- Неограниченное количество сервисов

⚠️ **Ограничения:**
- Сервис "засыпает" после 15 минут бездействия
- Первый запрос после "сна" может занять 30-60 секунд
- Для постоянной работы без "сна" нужен платный план ($7/месяц)

## Проблемы с регистрацией?

### Не приходит письмо подтверждения

1. Проверьте папку "Спам"
2. Подождите 5-10 минут
3. Попробуйте запросить повторную отправку
4. Используйте другой email

### Не могу подключить GitHub

1. Убедитесь, что у вас есть аккаунт на GitHub
2. Проверьте, что вы авторизованы на GitHub
3. Попробуйте отключить и подключить заново

### Забыл пароль

1. На странице входа нажмите "Forgot password?"
2. Введите email
3. Проверьте почту и следуйте инструкциям
